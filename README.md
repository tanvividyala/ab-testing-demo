# A/B Testing Demo
This demo is an interactive way to learn what A/B Testing is through simulating a 30-day test on different shades of blue for a link color. 

In 2009, Google famously tested 41 shades of blue and found that a slightly more purple shade generated an extra $200 million in annual revenue. The original test was actually a multivariate test across multiple elements, not a simple A/B test. This simulator is a simplified demonstration for educational purposes. Read more about the original test [here](https://medium.com/@whystudying/multivariate-test-googles-41-shades-of-blue-846d2c5781a8)

Made to support learning for COGS 9 (Introduction to Data Science).
